# Vanya2h's Home Page

Welcome to the source code of my page. Hope it helps.

## Prerequisites

- Node.js >= 22
- pnpm 10.32.1+

## Getting Started

```bash
corepack pnpm install
pnpm dev
```

## Scripts

| Script        | Command            | Description                  |
| ------------- | ------------------ | ---------------------------- |
| `dev`         | `pnpm dev`         | Start the development server |
| `build`       | `pnpm build`       | Build for production         |
| `start`       | `pnpm start`       | Start the production server  |
| `check-types` | `pnpm check-types` | Run TypeScript type checking |
| `lint`        | `pnpm lint`        | Run ESLint                   |
| `lint:fix`    | `pnpm lint:fix`    | Run ESLint with auto-fix     |
| `typegen`     | `pnpm typegen`     | Generate React Router types  |
| `sort-pkg`    | `pnpm sort-pkg`    | Sort package.json keys       |
